# 인라인 코드 강조
백틱(`): 틸드(~)아래 있는 버튼
백틱을 사용한다~!

CSS에서 `background` 혹은   
`background-image` 속성으로 요소에 배경 이미지를 삽입할 수 있습니다.


# 블록(block)코드 강조
백틱기호 세번 + html + (코드) + 백틱기호 세번


```html
<a href="http://www.google.co.kr/"
target="_blank">GOOGLE</a>
```
